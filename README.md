# Анализ предметной области: Страховые компании
 
## Описание
Практическая работа по анализу предметной области страховых компаний.
Содержит UML-диаграммы и скрипты автоматизации.

Список использованных источников
Федеральный закон «Об организации страхового дела в Российской Федерации» от 27.11.1992 № 4015-1.

Банк России. Статистика страхового рынка. — URL: https://cbr.ru

Буч Г., Рамбо Д., Якобсон А. Язык UML. Руководство пользователя. — М.: ДМК Пресс, 2020.

Орлов С.А., Цимбалюк Б.Я. Технологии разработки программного обеспечения. — СПб.: Питер, 2021.

Заключение:
Анализ предметной области «Страховые компании» позволил выявить ключевые проблемы и предложить пути их решения с помощью автоматизации и современных ИТ-решений. Разработанные UML-диаграммы и скрипты могут быть использованы в качестве основы для проектирования информационной системы страховой компании.
